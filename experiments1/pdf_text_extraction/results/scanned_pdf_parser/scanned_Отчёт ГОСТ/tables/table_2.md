| Обозначение                   | Формула (для одного изображения)                              | Пояснение                                                                     |
|:------------------------------|:--------------------------------------------------------------|:------------------------------------------------------------------------------|
| IoU (Intersection-over-Union) | $IoU = \frac{|B_{pred} \cap B_{gt}|}{|B_{pred} \cup B_{gt}|}$ | Мера перекрытия, предсказанного Bounding-Box $B_{pred}$ и эталонного $B_{gt}$ |
| Precision                     | $P = \frac{TP}{TP + FP}$                                      | Доля предсказанных боксов, попавших в «правильные» (IoU ≥ τ).                 |
| Recall                        | $P = \frac{TP}{TP + FN}$                                      | Доля эталонных боксов, которые модель успешно нашла.                          |
| F1@50                         | $F1@50 = \frac{2PR}{P + R'}$ при IoU ≥ 0.5                    | Гармоническое среднее Precision и Recall при пороге IoU = 0.5.                |
