# Визуальный инструмент разметки с автоматической предразметкой

## Описание

Этот инструмент автоматически создает предварительную разметку используя наш парсер, визуализирует её на изображении PDF с цветными прямоугольниками, и позволяет редактировать прямо в интерфейсе.

## Запуск

```bash
cd experiments/metrics
streamlit run gui_annotation_visual.py
```

## Возможности

### 1. Автоматическая предразметка
- Нажмите кнопку "✨ Создать автоматическую разметку"
- Инструмент использует наш парсер для создания предварительной разметки
- Все элементы автоматически получают координаты, типы и содержимое

### 2. Визуализация
- Разметка отображается на изображении PDF
- Каждый тип элемента имеет свой цвет:
  - 🔴 **title** - Красный
  - 🟠 **header_1** - Оранжевый
  - 🟡 **header_2** - Темно-оранжевый
  - 🟢 **header_3** - Желтый
  - 🔵 **text** - Голубой
  - 🟣 **table** - Фиолетовый
  - 🟪 **image** - Розовый
  - И т.д.

### 3. Редактирование
- Кликните на элемент в списке для редактирования
- Измените содержимое, тип, координаты
- Сохраните изменения
- Удалите ненужные элементы

### 4. Добавление элементов
- Добавьте новые элементы вручную
- Укажите тип, содержимое, координаты
- Установите родительский элемент

## Рабочий процесс

1. **Загрузите PDF** файл
2. **Создайте автоматическую разметку** - нажмите "✨ Создать автоматическую разметку"
3. **Просмотрите визуализацию** - все элементы отмечены цветными прямоугольниками
4. **Исправьте ошибки**:
   - Откройте элемент для редактирования
   - Измените содержимое, тип, координаты
   - Сохраните изменения
5. **Добавьте недостающие элементы** вручную
6. **Сохраните разметку** в JSON файл

## Преимущества

✅ **Быстрый старт** - автоматическая предразметка экономит время  
✅ **Визуальная проверка** - видите все элементы на изображении  
✅ **Цветовая кодировка** - легко различать типы элементов  
✅ **Удобное редактирование** - исправляйте ошибки прямо в интерфейсе  
✅ **Объективность** - предразметка создается нашим парсером, но вы можете её редактировать  

## Легенда цветов

Все цвета отображаются в боковой панели. Каждый тип элемента имеет уникальный цвет для легкой идентификации.

## Советы

- **Проверьте автоматическую разметку** - она может содержать ошибки
- **Исправьте координаты** - автоматические координаты могут быть неточными
- **Проверьте типы элементов** - заголовки могут быть определены неправильно
- **Добавьте недостающие элементы** - парсер может пропустить некоторые элементы
- **Проверьте иерархию** - убедитесь, что parent_id установлены правильно
