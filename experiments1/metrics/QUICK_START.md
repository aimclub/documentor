# Быстрый старт: Ручная разметка документов

## Шаг 1: Запуск инструмента разметки

```bash
cd experiments/metrics
python manual_annotation_tool.py --input test_files_for_metrics/2412.19495v2.pdf --output annotations/2412.19495v2_annotation.json
```

## Шаг 2: Разметка элементов

В интерактивном режиме используйте команды:

- `add` - добавить новый элемент
- `list` - показать все размеченные элементы
- `edit N` - редактировать элемент номер N
- `save` - сохранить разметку
- `quit` - выйти с сохранением

### Процесс добавления элемента:

1. **Выберите тип элемента** из списка (header_1, text, table, image, etc.)
2. **Введите текстовое содержимое** (или оставьте пустым для изображений)
3. **Укажите номер страницы** (1, 2, 3, ...)
4. **Введите координаты bbox** [x0, y0, x1, y1] или нажмите Enter для пропуска
5. **Выберите родительский элемент** (или 0 для root)

## Шаг 3: Что размечать?

### Обязательно размечайте:

1. **Все заголовки** (title, header_1 - header_6)
   - Тип, текст, страница, координаты
   - Родитель (для вложенных заголовков)

2. **Все текстовые блоки** (text)
   - Тип, текст, страница, координаты
   - Родитель (ближайший заголовок)

3. **Все таблицы** (table)
   - Тип, подпись (если есть), страница, координаты
   - Родитель
   - **Важно**: Структуру ячеек можно добавить позже в JSON

4. **Все изображения** (image)
   - Тип, подпись (caption), страница, координаты
   - Родитель

### Порядок разметки:

Размечайте элементы **в порядке их появления** в документе:
- Сверху вниз
- Слева направо
- По страницам (1, 2, 3, ...)

Поле `order` устанавливается автоматически (0, 1, 2, ...).

## Шаг 4: Проверка

После разметки всех элементов:

1. Выполните `list` для просмотра всех элементов
2. Проверьте порядок (`order`)
3. Проверьте иерархию (`parent_id`)
4. Сохраните (`save`)

## Шаг 5: Доработка в JSON

После сохранения можно вручную отредактировать JSON файл:

- Добавить структуру таблиц (`metadata.table_structure.cells`)
- Исправить координаты
- Добавить дополнительные метаданные

```
> add

Введите координаты bbox: 100, 50, 500, 80

Выберите родителя: 0

> add
...

> list
Всего элементов: 5
  1. [header_1    ] order=  0 page= 1 parent=None
     Introduction
  2. [text        ] order=  1 page= 1 parent=elem_0001
     This is the introduction text...

✓ Разметка сохранена
```

## Советы

1. **Размечайте по страницам** - сначала все элементы на странице 1, потом на странице 2, и т.д.
2. **Используйте PDF-просмотрщик** параллельно для проверки координат
3. **Сохраняйте часто** - используйте команду `save` регулярно
4. **Проверяйте порядок** - поле `order` должно соответствовать порядку чтения документа
