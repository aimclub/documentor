# Графический инструмент для разметки документов

Веб-интерфейс на основе Streamlit для удобной ручной разметки документов.

## Установка

```bash
pip install -r requirements_gui.txt
```

Или установите зависимости вручную:
```bash
pip install streamlit PyMuPDF Pillow streamlit-drawable-canvas
```

## Запуск

### Визуальный инструмент с автоматической предразметкой (рекомендуется):
```bash
cd experiments/metrics
streamlit run gui_annotation_visual.py
```

**Особенности:**
- 🤖 Автоматически создает предварительную разметку используя наш парсер
- 🎨 Визуализирует разметку на изображении PDF с цветными прямоугольниками
- ✏️ Позволяет редактировать элементы прямо в интерфейсе
- 🎯 Разные цвета для разных типов элементов

### Основной инструмент (с canvas):
```bash
cd experiments/metrics
streamlit run gui_annotation_tool.py
```

### Упрощенный инструмент (без canvas, если есть проблемы):
```bash
cd experiments/metrics
streamlit run gui_annotation_tool_simple.py
```

Интерфейс откроется в браузере по адресу `http://localhost:8501`

**Примечание:** Если возникает ошибка с canvas, используйте визуальную версию `gui_annotation_visual.py`

## Возможности

### 1. Загрузка PDF
- Загрузите PDF файл через интерфейс
- Или выберите из списка файлов в `test_files_for_metrics/`

### 2. Просмотр документа
- Навигация по страницам (кнопки или слайдер)
- Отображение страниц PDF с возможностью масштабирования

### 3. Добавление элементов
- Выберите тип элемента из списка
- Введите текстовое содержимое
- Укажите номер страницы
- Установите родительский элемент (для иерархии)
- **Выделите область на изображении** - нарисуйте прямоугольник для автоматического заполнения координат bbox
- Или введите координаты вручную [x0, y0, x1, y1]

### 4. Управление элементами
- Просмотр всех размеченных элементов
- Фильтрация по типу и странице
- Редактирование элементов
- Удаление элементов

### 5. Сохранение
- Сохранение разметки в JSON файл
- Загрузка существующей разметки для продолжения работы

## Интерфейс

### Боковая панель
- **Загрузка файлов**: PDF и JSON разметки
- **Сохранение**: путь и кнопка сохранения
- **Статистика**: количество элементов, страниц, распределение по типам

### Основная область
- **Навигация**: кнопки и слайдер для переключения страниц
- **Просмотр PDF**: отображение текущей страницы
- **Форма добавления**: все поля для создания нового элемента
- **Список элементов**: таблица со всеми размеченными элементами

## Использование

1. **Загрузите PDF** через боковую панель
2. **Перейдите на нужную страницу** используя навигацию
3. **Добавьте элемент**:
   - Выберите тип
   - Введите содержимое
   - Укажите страницу и координаты
   - Выберите родителя (если нужно)
   - Нажмите "Добавить элемент"
4. **Проверьте элементы** в списке внизу
5. **Сохраните разметку** через боковую панель

## Советы

- **Координаты bbox**: 
  - Выделите область на изображении, нарисовав прямоугольник (режим "Прямоугольник")
  - Координаты автоматически заполнятся в форме
  - Можно скорректировать координаты вручную
- **Иерархия**: Сначала размечайте заголовки, потом текстовые блоки с указанием родителя
- **Порядок**: Элементы автоматически получают порядковый номер
- **Сохранение**: Сохраняйте часто, чтобы не потерять работу
- **Рисование**: Переключитесь в режим "Прямоугольник" для выделения областей, "Просмотр" для обычного просмотра

## Примеры использования

### Разметка одного файла
1. Запустите `streamlit run gui_annotation_tool.py`
2. Загрузите PDF файл
3. Размечайте элементы страница за страницей
4. Сохраните результат

### Продолжение разметки
1. Загрузите существующий JSON файл разметки
2. Продолжите добавление элементов
3. Сохраните обновленную разметку
