# Инструмент разметки на PyQt5

Графический инструмент для разметки документов с автоматической предразметкой и визуализацией.

## Установка

```bash
pip install -r requirements_qt.txt
```

## Запуск

```bash
python qt_annotation_tool.py
```

## Возможности

### Основные функции

1. **Загрузка PDF**
   - Кнопка "📂 Загрузить PDF" для выбора файла
   - Автоматическое отображение первой страницы

2. **Автоматическая разметка**
   - Кнопка "✨ Авторазметка" создает предварительную разметку используя наш парсер
   - Элементы визуализируются цветными прямоугольниками на странице
   - ID начинаются с `elem_0000`

3. **Навигация по страницам**
   - Кнопки "◀ Предыдущая" и "Следующая ▶"
   - Отображение текущей страницы

4. **Визуализация разметки**
   - Каждый тип элемента имеет свой цвет
   - Выделение выбранного элемента более толстой рамкой
   - Подписи с типом и ID элемента

5. **Редактирование элементов**
   - Вкладка "Элементы": список всех элементов с возможностью выбора
   - Вкладка "Добавить": форма для создания нового элемента
   - Редактирование и удаление выбранных элементов

6. **Сохранение разметки**
   - Кнопка "💾 Сохранить" сохраняет разметку в JSON файл
   - Файл сохраняется в папку `annotations/`

### Цветовая схема элементов

- **title**: Красный
- **header_1**: Оранжевый
- **header_2**: Темно-оранжевый
- **header_3**: Желтый
- **header_4**: Ярко-желтый
- **header_5**: Желто-зеленый
- **header_6**: Зеленый
- **text**: Голубой
- **table**: Фиолетовый
- **image**: Розовый
- **list_item**: Зелено-голубой
- **caption**: Розово-красный
- **formula**: Синий
- **link**: Ярко-зеленый
- **code_block**: Серый

### Использование

1. Запустите инструмент
2. Загрузите PDF файл
3. Нажмите "✨ Авторазметка" для создания предварительной разметки
4. Просмотрите разметку на странице - элементы отображаются цветными прямоугольниками
5. Кликните на элемент на странице для его выбора
6. Используйте вкладку "Добавить" для добавления новых элементов
7. Используйте кнопки "Редактировать" и "Удалить" для управления элементами
8. Сохраните разметку кнопкой "💾 Сохранить"
